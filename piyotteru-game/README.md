# ピヨってるやつ、いるー？ - README

このプロジェクトは、シンプルなゲーム「ピヨってるやつ、いるー？」を実装したものです。プレイヤーは、ひよこやにわとりを集めながら、様々なアイテムを取得し、ショップでアイテムを購入してゲームを進めていきます。

## 構成

- `public/index.html`: ゲームのHTML構造を定義するファイルです。ゲームのUIや要素が含まれています。
- `src/css/style.css`: ゲームのスタイルを定義するCSSファイルです。レイアウトやデザインに関するスタイルが含まれています。
- `src/js/Animal.js`: 動物に関連するクラスや機能を定義するJavaScriptファイルです。ゲーム内の動物の振る舞いを管理します。
- `src/js/Game.js`: ゲームのロジックを管理するクラスを定義するファイルです。ゲームの進行や状態管理を行います。
- `src/js/ui.js`: UIの更新やイベント処理を管理するファイルです。ユーザーインターフェースの操作を担当します。
- `src/js/mikey.js`: マイキーの振る舞いを管理するファイルです。マイキーのアニメーションやセリフを制御します。
- `src/js/items/Meat.js`: 肉アイテムを定義するクラスを含むファイルです。肉を取得した際の処理や進捗バーの管理を行います。
- `src/js/items/index.js`: アイテム関連のエクスポートをまとめるファイルです。肉アイテムや他のアイテムを管理します。
- `src/js/shop/Shop.js`: ショップの機能を管理するクラスを定義するファイルです。アイテムの購入や管理を行います。
- `src/js/shop/shopItems.js`: ショップで販売されるアイテムのリストや効果を定義するファイルです。ゲーム攻略に役立つアイテムを管理します。
- `src/js/utils/helpers.js`: ユーティリティ関数を定義するファイルです。共通の処理やヘルパー関数が含まれています。
- `src/types/index.d.ts`: TypeScriptの型定義を含むファイルです。プロジェクト内で使用される型を定義します。
- `package.json`: プロジェクトの依存関係やスクリプトを管理するファイルです。npmの設定が含まれています。
- `.gitignore`: Gitで無視するファイルやディレクトリを指定するファイルです。

## ゲームの特徴

- プレイヤーは肉を集め、タップすることで肉を消費できます。
- 肉を取得した数はUI上に表示され、進捗バーが進みます。
- 進捗バーが最大に達すると肉は消去されますが、取得件数には含まれません。
- 肉を集めることでショップでアイテムを購入でき、ゲーム攻略に役立つ様々な効果を持つアイテムが用意されています。
- 例えば、オオカミを放つことで一定時間経過後に無作為にニワトリを食い殺す機能があります。

## インストール

1. リポジトリをクローンします。
2. 必要な依存関係をインストールします。
   ```
   npm install
   ```
3. 開発サーバーを起動します。
   ```
   npm start
   ```
